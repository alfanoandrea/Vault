Introduzione

La crittografia è una tecnica utilizzata per proteggere le informazioni trasformandole in un formato illeggibile per chiunque non possieda la chiave per decrittarle. È uno strumento fondamentale per garantire la riservatezza e l'integrità dei dati in molte applicazioni, dalle comunicazioni online alle transazioni finanziarie.

Esistono due principali tipi di crittografia: simmetrica e asimmetrica. Nella crittografia simmetrica, la stessa chiave viene utilizzata sia per cifrare che per decifrare i dati. Nella crittografia asimmetrica, vengono utilizzate due chiavi diverse: una chiave pubblica per cifrare e una chiave privata per decifrare.



Algoritmi di Crittografia

AES (Advanced Encryption Standard):
AES è uno degli algoritmi di crittografia simmetrica più utilizzati al mondo. È veloce ed efficiente, e viene utilizzato in molte applicazioni di sicurezza informatica, come il cifratura di file e la protezione delle comunicazioni online.

RSA (Rivest-Shamir-Adleman):
RSA è un algoritmo di crittografia asimmetrica ampiamente utilizzato per la cifratura di dati sensibili e la firma digitale. Utilizza una coppia di chiavi, una pubblica e una privata, per garantire la sicurezza delle informazioni.

DES (Data Encryption Standard):
DES è un algoritmo di crittografia simmetrica che, nonostante la sua obsolescenza, ha gettato le basi per molti altri algoritmi di crittografia moderna.

Blowfish:
Blowfish è un algoritmo di crittografia simmetrica noto per la sua velocità e sicurezza. È spesso utilizzato per proteggere dati in applicazioni software e hardware.

ECC (Elliptic Curve Cryptography):
ECC è un algoritmo di crittografia asimmetrica che offre un alto livello di sicurezza con chiavi di dimensioni più ridotte rispetto ad altri algoritmi, come RSA.



Esempi di Crittografia e Decrittografia

Crittografia con AES:
Immagina di voler cifrare il messaggio "Ciao Mondo" utilizzando AES con una chiave segreta. Ecco un esempio di come potrebbe essere fatto in Python:


                    
    import base64
    from Crypto.Cipher import AES
    from Crypto.Util.Padding import pad, unpad
    
    # Chiave e messaggio
    chiave = b'LaChiaveSegreta!'
    messaggio = b'Ciao Mondo'
    
    # Cifratura
    cipher = AES.new(chiave, AES.MODE_CBC)
    ct_bytes = cipher.encrypt(pad(messaggio, AES.block_size))
    iv = base64.b64encode(cipher.iv).decode('utf-8')
    ct = base64.b64encode(ct_bytes).decode('utf-8')
    print(f"IV: {iv}")
    print(f"Testo Cifrato: {ct}")
        
    # Decrittografia
    iv = base64.b64decode(iv)
    ct = base64.b64decode(ct)
    cipher = AES.new(chiave, AES.MODE_CBC, iv)
    messaggio_decifrato = unpad(cipher.decrypt(ct), AES.block_size)
    print(f"Messaggio Decifrato: {messaggio_decifrato.decode('utf-8')}")

In questo esempio, il messaggio "Ciao Mondo" viene cifrato e decifrato utilizzando AES in modalità CBC.

Crittografia con RSA:
Ecco un esempio di come cifrare e decifrare un messaggio utilizzando RSA in Python:


    from Crypto.PublicKey import RSA
    from Crypto.Cipher import PKCS1_OAEP
    import base64
    
    # Genera una coppia di chiavi RSA
    chiave = RSA.generate(2048)
    chiave_privata = chiave.export_key()
    chiave_pubblica = chiave.publickey().export_key()
    
    # Cifratura
    cipher = PKCS1_OAEP.new(RSA.import_key(chiave_pubblica))
    messaggio = b'Ciao Mondo'
    ct = base64.b64encode(cipher.encrypt(messaggio)).decode('utf-8')
    print(f"Testo Cifrato: {ct}")
