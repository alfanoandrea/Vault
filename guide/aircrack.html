Introduzione

Aircrack-ng è una suite di strumenti progettati per la valutazione della sicurezza delle reti Wi-Fi. Questa guida fornisce una panoramica su come installare e utilizzare Aircrack-ng per testare la sicurezza delle reti wireless.


Installazione

Per installare Aircrack-ng, utilizza i seguenti comandi, a seconda della tua distribuzione Linux: 


    # Ubuntu
    sudo apt update
    sudo apt install aircrack-ng

    # Fedora
    sudo dnf install aircrack-ng


Modalità Monitor

Per utilizzare Aircrack-ng, è necessario mettere la scheda di rete in modalità monitor con il comando: 


    sudo airmon-ng start wlan0

*Nota: Sostituisci wlan0 con il nome della tua interfaccia di rete.*


Monitoraggio delle Reti

Utilizza airodump-ng per scansionare le reti Wi-Fi disponibili: 


    sudo airodump-ng wlan0mon


Cattura dei Pacchetti

Dopo aver individuato una rete, puoi catturare i pacchetti utilizzando: 


    sudo airodump-ng --bssid [BSSID] -c [CANALE] -w [FILE] wlan0mon

*Dove [BSSID] è l'indirizzo MAC del router, [CANALE] è il canale della rete e [FILE] è il nome del file di output.*


Attacco WEP e WPA/WPA2

Per attaccare reti WEP, utilizza: 


    sudo aireplay-ng -1 0 -e [ESSID] -a [BSSID] wlan0mon

Dopo aver catturato abbastanza pacchetti, utilizza aircrack-ng per decifrare la chiave: 


    aircrack-ng [FILE].cap

Per reti WPA/WPA2, cattura l'handshake e utilizza un file di dizionario: 


    aircrack-ng -w [DIZIONARIO] [FILE].cap


Etica e Legalità

È fondamentale utilizzare Aircrack-ng solo su reti per le quali hai ricevuto il permesso di testare la sicurezza. Accedere a reti senza autorizzazione è illegale e contro l'etica.


Conclusione

Aircrack-ng è uno strumento potente per la valutazione della sicurezza delle reti Wi-Fi. Utilizzalo in modo responsabile per migliorare le tue competenze nel campo della cybersecurity.
